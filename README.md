# Frontend for Screetner

# How to run this project

1. Clone the repository
2. Install dependencies using `npm install`
3. Create a `.env` file in the root directory and add values following the `.env.example` file
4. Run the project using `npm run dev`

### This project no longer has a `Dockerfile` because the website will be deployed on `Heroku`. If you need one, please create it yourself.
